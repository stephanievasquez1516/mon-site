système d'alarme avec capteur ultrason

Développement d'un système de sécurité basé sur Arduino. Le capteur ultrason (HC-SR04) détecte une intrusion et déclenche une alarme. Un clavier de 16 touches permet d'entrer un code à 4 chiffres pour activer/désactivé le système. Une DEL bicolore indique l'état: armé, désarmé ou alarme active.

  • Technologies: Arduino, capteur HC-SR04, clavier matriciel, DEL bicolore
  • Fonctionnalités:Détection intrusion, alarme visuel, code de désactivation
